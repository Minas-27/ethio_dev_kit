# Changelog

## 0.1.0

Initial release.

- `holidaysForYear(int gregorianYear)` computes fixed holidays, Orthodox moveable feasts, and tabular Islamic holiday estimates dynamically.
- `calculateOrthodoxEaster(int year)` public utility for Meeus' Julian Easter computus.
- `tabularHijriToGregorian(int year, int month, int day)` public utility for civil Islamic calendar estimation.
