# ethiopic_typography example

A minimal Flutter app showing `ethiopic_typography` in action:

- Mixed **English + Amharic** text across every Material text role.
- A **light / dark** mode toggle (app-bar icon), both driven by
  `ethioAppTheme()` / `ethioDarkAppTheme()`.
- A one-off bilingual style via `EthioTextTheme.style()`.

## Run

```bash
flutter run
```
